Surfin

Statistical Inference for Random Forests

Download .zip Download .tar.gz View on GitHub

Description

This R package computes standard error estimates for random forest predictions using a fast implementation of random forests in C++. Two variance estimates are provided: U-statistic based and infinitesimal jackknife.

Latest Version

v0.0.0.9000, under development

Installation

Package not yet on CRAN. For now, download from this website the .zip (if you are on Windows) or .tar.gz (if you are on Mac / Linux). If you don't already have the dependencies (Rcpp, RcppArmadillo, Matrix) installed, install those from CRAN first. Then type the following in R:

$ install.packages(path_to_downloaded_file, repos=NULL, type="source")
$ library(surfin)
