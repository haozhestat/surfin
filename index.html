Surfin

Statistical Inference for Random Forests

Download .zip Download .tar.gz View on GitHub

Description

This R package computes uncertainty for random forest predictions using a fast implementation of random forests in C++. Two variance estimates are provided: U-statistic based (Mentch & Hooker, 2016) and infinitesimal jackknife (Wager, Hastie, Efron, 2014).

The latter is also available from the authors' R package randomForestCI.

Check out a demo of surfin on some real data! How Uncertain Are Your Random Forest Predictions?

Installation

Package not yet on CRAN. For now, download from this website the .tar.gz (whether you are on Mac, Linux, or Windows). Do not download the .zip file. Note that both are source, not binary files. If you don't already have the dependencies (Rcpp, RcppArmadillo, Matrix, knitr) and optional dependencies (randomForest, rpart) installed, install those from CRAN first. If you are on Windows, make sure you have C++ development tools installed (RTools). If you already have an older version of surfin installed, remove that first by typing the following in R: 

$ remove.packages("surfin")
Then install using:

$ install.packages(path_to_downloaded_file, repos=NULL, type="source")
$ library(surfin)
Learn how to use surfin from:

$ ?surfin
