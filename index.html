Surfin

Statistical Inference for Random Forests

View on GitHub Download .zip Download .tar.gz

Description

This R package computes standard error estimates for random forest predictions using a fast re-implementation of random forests in Rcpp. Two kinds of standard error estimates provided: U-statistics based and infinitesimal jackknife.

Latest Version

v0.0.0.9000, under development

Installation

Type the following in R to obtain the package from CRAN:

$ install.packages("surfin")

Or download from this website the .zip (if you are on Windows) or .tar.gz (if you are on Mac / Linux) then type the following in R:

$ install.packages(path_to_downloaded_file, repos=NULL)
